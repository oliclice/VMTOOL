# VM-TOOL 代码审查检查清单

## 使用说明

本检查清单用于VM-TOOL项目的代码审查过程，涵盖五个审查阶段的所有检查项。每个检查项使用复选框标记完成状态，并提供问题描述和修复建议字段。

**使用指南**:
1. 按阶段顺序执行审查
2. 对每个检查项进行检查并标记状态
3. 发现问题时记录在"问题描述"列
4. 提供具体的修复建议
5. 完成后生成审查报告

**状态标记**:
- ✅: 通过检查
- ⚠️: 存在问题，需要修复
- ❌: 严重问题，必须修复
- 🔄: 进行中
- N/A: 不适用

### A. 工具命令参考

```bash
# 静态分析
mypy --strict app/
ruff check --fix .
black --check .
isort --check-only .
bandit -r app/ -f json
pip-audit -r requirements.txt
radon cc app/ -s

# 测试
pytest -v tests/
pytest --cov=app --cov-report=html
pytest --benchmark-only
```
